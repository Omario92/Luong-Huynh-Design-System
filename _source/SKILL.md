# Luong Portfolio Design Skill

## Purpose

Create a premium portfolio website for Luong Huynh, a Digital Artist specializing in 3D visuals, AI-generated imagery and VFX.

The output must be easy to translate into Elementor, not only visually impressive in a design canvas.

## Source positioning

Use this positioning as the source of truth:

> Luong Huynh is a Digital Artist specializing in 3D, AI-generated Art and VFX, based in Ho Chi Minh City.

Avoid positioning him primarily as a fullstack developer unless the user explicitly asks for that direction.

## Design direction

Default direction: **Cinematic Digital Artist Portfolio**.

Visual language:

- Dark cinematic base
- Premium black / charcoal background
- Warm ivory text
- Acid-lime or amber accent, used sparingly
- Large editorial typography
- High-impact image/video cards
- Minimal UI chrome
- Smooth reveal interactions
- Strong whitespace and grid discipline

The website should feel like a visual studio, not a generic agency template.

## Homepage structure

Generate the homepage using this sequence:

1. Header
   - Logo/name: Luong Huynh
   - Navigation: Homepage, Portfolio, About
   - Optional CTA: Available for Work

2. Hero
   - Main title: Luong Huynh
   - Subtitle: Digital Artist crafting 3D visuals, AI-generated imagery and VFX for brands, campaigns and cinematic stories.
   - Metadata: Ho Chi Minh City · 2026
   - CTA buttons: View Portfolio, Get in Touch
   - Hero media: one large cinematic visual or a grid of selected works

3. Selected Works
   - Feature Stride Beyond
   - Feature Nexora
   - Feature BT Studio – CGI Beverage Demo
   - Keep cards media-first

4. Capabilities
   - 3D Product & CGI Visuals
   - AI-generated Campaign Imagery
   - VFX, Compositing & Post-production
   - Visual Direction for Brand Campaigns

5. Portfolio teaser
   - Mention categories: CGI, AI Generated, 3D Model, Key Visuals, TVC, Branding

6. About teaser
   - Short human copy focused on craft, technology and brand collaboration

7. Contact CTA
   - Available for Work
   - Behance / Dribbble / Instagram / LinkedIn

## About page correction

Do not use generic web-development copy such as Responsive Design, eCommerce, Maintenance and Development unless the user asks to include web development services.

Replace with:

- 3D Product & CGI Visuals
- AI-generated Campaign Imagery
- VFX, Compositing & Post-production
- Visual Direction for Brand Campaigns

Recommended about copy:

> I’m a digital artist focused on 3D visuals, AI-generated images and VFX. I collaborate with brands, agencies and production teams to create campaign-ready visuals, cinematic key visuals and experimental AI-assisted imagery.

## Elementor handoff rules

Always provide:

- Section breakdown
- Elementor widget/container mapping
- Class names
- CSS variables
- Responsive notes
- Motion/interactions
- What should be built with Elementor controls vs custom CSS

Use class prefix `lh-` for all custom classes.

Do not produce overly complex code that is hard to maintain inside Elementor.

## CSS rules

- Use CSS custom properties for colors, radius, spacing and type scale.
- Prefer class-based selectors.
- Avoid targeting unstable Elementor-generated IDs.
- Keep animations lightweight.
- Ensure mobile layout is not an afterthought.
- Respect reduced-motion preferences.

## Quality checklist

Before finalizing a design, check:

- Does the homepage immediately say 3D / AI / VFX?
- Does the design feel cinematic and premium?
- Are project images/videos the hero, not decoration?
- Are About and Services consistent with Digital Artist positioning?
- Can an Elementor user build it without custom JavaScript?
- Are the classes reusable across pages?
- Is the mobile experience clean?
